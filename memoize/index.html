✅ First Topic: Memoization

🔹 Why do we need Memoization?

Sometimes you call the same function again and again with the same input, and it repeats heavy calculations unnecessarily. Memoization helps you cache the result so you don’t recalculate it.

🔹 What is Memoization?

Think of it like saying:

“If I’ve already answered this question before, I’ll just give you the same answer instantly — no thinking needed!”

🔹 Real-Life Analogy You're asked, “What’s 12 x 12?” — first time you calculate it. Next time, you remember it’s 144. That’s memoization.
