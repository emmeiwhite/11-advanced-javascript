⚡️ What is Debouncing?

Debouncing is a technique to delay a function call until the user has stopped performing an action for a set period of time.

🔹 WHY do we need it?

Imagine a user is typing in a search bar. Without debouncing:

  • You fire an API request on every keystroke (wasteful, heavy on performance).

    • With debouncing:

  • You wait until the user stops typing for, say, 500ms, and then fire the request.

    • 📱 Real-World Analogy

    You call someone’s name, they don't respond immediately — they wait a few seconds to ensure you’re not calling again. That’s debouncing.
